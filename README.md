# Tugas-Rekrusif

Repositori ini berisi implementasi beberapa masalah klasik rekursif dan backtracking menggunakan Python, yaitu N-Queens (N-Ratu), Knight’s Tour (Tur Kuda), dan Knapsack Problem. Program dibuat untuk menunjukkan bagaimana algoritma rekursif dapat digunakan untuk mencari solusi secara otomatis pada berbagai permasalahan.

---

# 1. N-Queens (N-Ratu)

Program N-Queens digunakan untuk menempatkan beberapa ratu pada papan catur tanpa saling menyerang. Setiap ratu tidak boleh berada pada baris, kolom, maupun diagonal yang sama. Program menggunakan algoritma backtracking rekursif untuk mencoba berbagai kemungkinan posisi hingga menemukan solusi yang valid.

---

# 2. Knight’s Tour (Tur Kuda)

Program Knight’s Tour digunakan untuk mencari jalur langkah kuda agar semua petak pada papan catur dikunjungi tepat satu kali. Algoritma backtracking rekursif digunakan untuk mencoba setiap kemungkinan langkah kuda sampai ditemukan jalur yang memenuhi syarat.

---

# 3. Knapsack Problem

Program Knapsack digunakan untuk mencari kombinasi barang agar total berat sesuai dengan target yang ditentukan. Setiap barang dapat dipilih atau tidak dipilih. Program akan mencoba berbagai kombinasi menggunakan rekursi hingga menemukan solusi yang cocok.
